# **App Name**: Momentum

## Core Features:

- User Authentication: Secure user authentication system with email/password and Google OAuth login.
- Habit Tracking: Create, track, and manage daily/weekly habits with progress visualization and reminders.
- Routine Management: Create and follow routines consisting of a series of habits to complete sequentially.
- Task Management: Create, organize, and manage tasks with due dates, subtasks, notes, and priority levels.
- Goal Setting: Set and track progress towards long-term goals with visual progress indicators.
- Mood Tracking: Record daily mood levels with emojis and notes, visualize mood trends over time.
- Financial Tracking: Manage income/expenses, track budgets, and visualize spending patterns.

## Style Guidelines:

- Primary color: A muted, contemplative blue (#6699CC) to evoke calmness and focus.
- Background color: A light, desaturated blue-gray (#F0F4F8) for a clean, modern backdrop.
- Accent color: A warm, contrasting orange (#E67E22) to highlight key actions and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable interface.
- Use a set of consistent and minimalist icons, potentially from a library like Material Design Icons or Font Awesome, to represent different features and actions.
- Employ a clean and structured layout, using a grid system to maintain consistency and responsiveness across different screen sizes.
- Incorporate subtle animations and transitions to enhance user experience and provide feedback, such as button hover effects, loading animations, and smooth page transitions.
